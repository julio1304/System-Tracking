Time Clock Web Application


-Using Blazor, C#, HTML, CSS, JavaScript
The application is divided into two roles: employee and admin.
The employee role will have access to TimeClock Page and Employee-Timesheet page.
The admin role will have additional access to Admin-Employee, Admin-Timesheet, and Admin-Export/Delete.

Time Clock Homepage

Purpose: Employees will clock in and clock out by entering four digit unique employee ID created by the admin.
Today's Status will provide live update of today's activity.

Employee-Timesheet Page

Purpose: Employees can check their worked hours by typing their employee ID and selecting date range.
At the bottom, the program will auto-calculate total hours worked based on the date range.

Login Page

Purpose: Admin can gain access by logging in.

Admin-Employee Page

Purpose: Admin can view registered employees. For the employees to use this application, they must be registered on the Admin-Employee page.

CRUD Employee

Purpose: Admin can add/edit/delete employee.

CRUD Employee

Purpose: Admin can add/edit/delete employee.
